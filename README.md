# 광운대 강의계획서 검색기

> 광운대학교 학부 강의 계획서를 조회할 수 있는 서비스입니다.  
> 로그인이 필요합니다.  

## 로컬에서 실행하기

#### 1. Node.js 설치
아래 홈페이지에서 Node.js를 설치합니다.  
https://nodejs.org/en/download/ 

#### 2. 강의계획서 검색기를 클론합니다.
```
$ git clone https://github.com/0420syj/kw-syllabus
```

#### 3. 프로젝트 실행에 필요한 패키지를 설치합니다.
```
$ cd kw-syllabus
$ npm install
```

#### 4. 프로젝트를 실행합니다.
```
$ npm start
```

#### 5. 브라우저 실행 후 localhost:3000으로 접속합니다.
